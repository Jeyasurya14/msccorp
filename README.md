# Michael Scott Contractor LLC Website

Professional corporate website for Michael Scott Contractor LLC, built with Next.js and Tailwind CSS.

## Features
- **Modern UI/UX**: Responsive design, custom "Dark Blue & Gold" theme.
- **Service Showcases**: Detailed pages for Retail Remodels, Mall Construction, and Project Management.
- **Contact Form**: Secure, validated contact form with email notifications (powered by Resend).
- **SEO Optimized**: Metadata and structure ready for search engines.
